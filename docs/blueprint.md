# **App Name**: Pal Foundation

## Core Features:

- Ambulance Request Submission: Allow users to submit ambulance requests via a form, which is stored in Firestore.
- Animal Adoption Listings: Dynamically display adoptable animals from Firestore in a grid layout.
- Volunteer Sign-Up: Allow users to sign up as volunteers, storing their information in Firestore.
- Contact Form Submission: Enable users to send messages via a contact form, with submissions saved to Firestore.
- Success Stories Display: Showcase success stories dynamically fetched from Firestore.
- Animal Description Enrichment Tool: When a new animal's data is entered, this AI tool is automatically triggered, reads that data, and proposes additional descriptive text about the animal to entice potential adopters. A user can choose to accept or reject the suggestions. This is an AI tool.
- Razorpay Donation Handling: Integrate Razorpay for handling donations, including UPI and QR code payment options.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke compassion and tranquility.
- Background color: Light beige (#F5F5DC), a very low saturation of the primary to keep a bright and airy mood.
- Accent color: Muted rose (#D87093) to highlight calls to action without overwhelming the design.
- Body and headline font: 'PT Sans', a sans-serif offering a balanced modern yet warm aesthetic.
- Use simple, friendly icons that complement the theme of animal welfare.
- Implement responsive grid layouts using Tailwind CSS to ensure seamless adaptation across devices.
- Incorporate subtle animations using framer-motion to enhance user experience and engagement.
